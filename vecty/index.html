HTTP Request from inside WASM using Vecty frontend library

This example uses Vecty UI library for Golang.

See the output in browser developer console.

Actual code:

func (b *Body) httpRequest(*vecty.Event) {
  dispatcher.Dispatch(&actions.HttpRequest{Url: "https://httpbin.org/anything"})
}
